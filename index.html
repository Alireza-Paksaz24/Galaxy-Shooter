4. Create the player

1. in Main camera, you could change background color. the default setting for Clear Flags is Skybox.

2. one of the best way to apply movement and speed, is Translate method in transfor class. this method get one input and that is a vector for new position. you could use this input : Vector3 * speed * Time.deltatime.

3. GetAxis, is a method from Input class that return 0 till 1 float number that shows how much player press the button. GetKey, is a method that return boolean, shows if player press the key or not.

6. Enemy

1. we can use Random class from MonoBehavior and Range for creating rendom float number (float A = Random.Range(10.0f,20.0f);)

2. There is 2 type of collision. hard surface collision and trigger. the hard surface happend when 2 object are collide with eachother. like 2 car crashed. but trigger is when 2 object detect that they are collide but it doesn't affact the movement. like collecting the coin in super Mario.

3. How to create a tag? at the top of inspector view, select 'tag' and choose 'add tag'. then, you can use 'other.tag' in the OnTerrigerEntr to understand what is object.

4. How to get component? other.transform.GetComponent<component>()

5. The GameObjcet who wants to detect collition have to have a rigidbody.

7. Spawn Manager

1. This course show us how to use coroutine to have a routin in our game such as spawn Enemy. this is how you could use a coroutin.

1.1. first of all, we need a method with IEnumerator method. this kind of method can have special return type called 'yield return'. this return type means that this methid stops for amount of time and start continue other stuff. and after times passed, game return to this method and do the rest of method.

1.2. return type of IEnumerator method: new WaitFor...();

2. We can use GameObjcet.Find to find a object in game view. example: GameObjcet.Find("Player").GetComponent()

3. how to set parents of GameObject? using {Game-Object-Name}.transform.parents
